VSCode HTML Eklentileri

VSCode Eklentileri Ne İşe Yarar?

Visual Studio Code eklentileri, modern web geliştirmede çok önemlidir. Visual Studio Code eklentileri, geliştirme iş akışınızı desteklemek için kurulumunuza hata ayıklayıcılar, diller ve araçlar eklemenize olanak tanır.

Eklentiler

HTML CSS Support:

  • HTML içinde CSS sınıf ve ID önerileri sunar ve CSS dosyalarınıza kolayca bağlantı eklemenizi sağlar.

Live Server:

  • Canlı bir önizleme sunarak HTML dosyalarınızı anında tarayıcıda görüntülemenizi sağlar.

Auto Rename Tag:

  • Bir HTML etiketini yeniden adlandırdığınızda eşleşen kapanış etiketini otomatik olarak günceller.

Auto Close Tag:

  • Bir HTML etiketi yazdığınızda otomatik olarak kapanış etiketini ekler.

Prettier:

  • HTML ve diğer kodları otomatik olarak biçimlendirir ve düzenler.

Indent Rainbow:

  • HTML içinde birbiriyle eşleşen tagları renkleri kullanarak daha temiz ve okunaklı hale getirir.

Better Comments:

  • Yorumları kategorize ederek daha okunaklı ve efektif bir kod yazımı sağlar.

CSS Peek:

  • HTML dosyası içinde CSS ID'si üzerine gelindiğinde peek atarak CSS içeriğini öngörmeyi sağlar.

Path intellisense:

  • Visual Studio Code içerisinde kod içinde dosya dizini kullanılacağında dizini kendi tamamlayarak yardımcı olur.
